# Qualykids Tech Stack Analysis

## Source Site

- WordPress 6.x.
- Kidscare theme.
- Slider Revolution for the hero.
- WPBakery / Visual Composer CSS assets.
- Google Fonts: Source Sans Pro, Signika, Fredoka One, Ubuntu.
- Fontello icons.

## Clone Implementation

- Next.js 16 App Router.
- React 19.
- Tailwind CSS v4 plus custom component classes in `globals.css`.
- `next/font/google` for Source Sans 3, Signika, and Fredoka.
- Local assets saved under `public/images/qualykids`.
- Lucide React icons for footer contact markers and play icon.
