# Hero Component Spec

Target: Qualykids home hero.

## Structure

- Outer yellow full-width section.
- Fixed-height stage: 633px desktop, 430px mobile.
- Absolute-positioned PNG layers:
  - sun behind child
  - child on left
  - three hanging clouds
  - UFO on right
  - text block on right
  - slider pagination dot

## Desktop Measurements

- Stage max width: 1425px.
- Text starts near x 665 and y 365 in the full-page screenshot.
- Hero text color: `#033761`.
- Hero type: Fredoka-style rounded display.
- Main text size: 40px.

## Mobile Behavior

- The original keeps a wider hero composition and crops it inside the viewport.
- CTA inside the hero is hidden on mobile; the separate pink appointment strip remains visible.
