# Qualykids Behavior Notes

Target inspected: `https://www.qualykids.com.br/`

## Scroll

- No meaningful sticky header behavior was observed on the home screenshot path used for the clone.
- The visible landing is mostly static. The original is a WordPress Kidscare theme with Revolution Slider assets, but only one slide state was visible during capture.

## Click

- Header links navigate to WordPress pages.
- `AGENDAMENTO` and the CTA buttons navigate to `https://clinic.globalhealth.mv/agenda-online/c/qualykids`.
- Video thumbnail links to the YouTube video `2WJhc_G8IKU`.

## Hover

- Header nav hover shifts to the pink accent used by the theme.
- Video thumbnail keeps a centered circular play affordance.

## Responsive

- Desktop: horizontal logo/nav header, 633px hero.
- Mobile: centered logo, separated `MENU` text, hero composition remains wider than viewport and is cropped like the original.
- Mobile footer stacks visually with larger text and keeps the video thumbnail constrained to the available width.
